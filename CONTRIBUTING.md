# Contributing

## Reporting broken selectors

Dreamina updates their UI occasionally, which may break the scraper. If something stops working:

1. Open an issue with a screenshot of the broken UI element
2. Include the error message from the terminal
3. If you can find the correct selector in DevTools, include it — that speeds things up a lot

## Pull requests

- Open an issue first to discuss the change
- Keep PRs focused — one fix or feature per PR
- All code and comments in English

## What's useful to contribute

- Bug fixes when Dreamina updates their UI
- Better selector strategies that are more resilient to UI changes
- Support for other Dreamina generation modes (AI Image, AI Avatar, etc.)

## What's out of scope

- Features that require a higher-tier Dreamina plan than Basic (Standard, Ultra)
- Features that bypass Dreamina's rate limits aggressively
